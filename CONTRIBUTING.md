# Contributing to Xi

Thank you for your interest in Xi! This is an early-stage research project.

## Areas of Contribution

1. **Formalization** — Proving properties of the type system in Lean 4 or Coq
2. **Implementation** — Building a fast type checker and graph reducer
3. **Hardware** — FPGA prototyping of reduction cores (see Reduceron/Heron projects)
4. **Standard library** — Encoding common data structures as Xi graphs
5. **Tooling** — Debuggers, visualizers, profilers for Xi graphs
6. **Documentation** — Tutorials, translations, examples

## Setup

```bash
git clone https://github.com/maja0027/xi-lang.git
cd xi
python src/xi.py demo
```

Requires Python 3.10+.

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make changes with clear commit messages
4. Add tests for new functionality
5. Submit a pull request
